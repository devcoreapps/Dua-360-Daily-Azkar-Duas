# Dua 360 - Daily Azkar Duas Privacy Policy

Public privacy policy page for the **Dua 360 - Daily Azkar Duas** app, hosted via GitHub Pages.

- Live page: `index.html`
- Source content: `Privacy Policy.md`

## Hosting

This repo is deployed with GitHub Pages from the `main` branch (root). Once Pages is enabled, the policy is available at:

```
https://devcoreapps.github.io/Dua-360-Daily-Azkar-Duas/
```

Use this URL in app store listings (Google Play / App Store) wherever a privacy policy link is required.
